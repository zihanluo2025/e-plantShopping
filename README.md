# Paradise Nursery – React Shopping Application

Paradise Nursery is a React-based web application designed to simulate an online plant nursery shopping experience.
The application allows users to explore plants, add items to a shopping cart, and manage their purchases through a clean and intuitive user interface.

# Tech Stack
- React – UI development

- JavaScript (ES6+)

- React Router – client-side routing

- CSS – styling and layout

- Vite – development and build tooling

# How to Run the Project

## Clone the repository:

`git clone https://github.com/your-username/paradise-nursery.git` 

## Install dependencies:

`npm install`

## Start the development server:
`npm run dev`
